# Invite users

Application invite users and display them in a list

### Installation

```sh
$ cd invite-users
$ npm install
```
### Runing the app
Run `npm run start:all` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
